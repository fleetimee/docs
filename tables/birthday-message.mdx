## Migration DDL

#### DDL

Script DDL untuk membuat tabel `birthday_message`.

```sql
CREATE TABLE birthday_message
(
    id         uuid      DEFAULT gen_random_uuid() NOT NULL
        CONSTRAINT birthday_message_id_unique
            PRIMARY KEY,
    idol_id    varchar(10)                         NOT NULL
        CONSTRAINT birthday_message_idol_id_idol_id_fk
            REFERENCES idols
            ON UPDATE CASCADE ON DELETE CASCADE,
    message    varchar                             NOT NULL,
    created_at timestamp DEFAULT NOW()             NOT NULL,
    updated_at timestamp DEFAULT NOW()             NOT NULL
);

ALTER TABLE birthday_message
    OWNER TO doadmin;
```

## Columns

#### id

- **Type:** `string`
- **Required:** `true`
- **Description:** ID UUID unique dari data ucapan ulang tahun.

```bash

  "id": "f7b3b2b3-4b1b-4b1b-4b1b-4b1b4b1b4b1b",
}
```

#### idol_id

- **Type:** `string`
- **Required:** `true`
- **Description:** ID Idol yang diambil dan berelasi dengan tabel `idols`.

```bash
{
  "idol_id": "JKT48-1",
}
```

#### message

- **Type:** `string`
- **Required:** `true`
- **Description:** Pesan ucapan ulang tahun yang akan dikirimkan.

```bash
}
```
